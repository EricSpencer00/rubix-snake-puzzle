04Formal proofs

This is the first formal verification of the Rubik's Snake in any proof assistant. Three complementary approaches cover different verification depths.

Coq / Rocq

State space formalization

Wedge geometry, rotation semantics, collision predicate, symmetry proofs. Machine-checked test vectors for n=2..14.
TLA+

Model checking

Incremental state machine spec. Invariant: no cell occupied twice. Configurable N for TLC exhaustive checking.
Python

Reference enumerator

Backtracking search with collision pruning. Validated test vectors cross-check the Coq proofs.
